Counting with a local variable.

The underlying principle is: avoid global variables, as they are available
to the entire scope of the containing window and may be changed by another piece of code in the same window.

0

A closure is a function that has access to the parent scope,
even after the parent function has closed.

Use only when really necessary as the code is a bit obscure to read.

The way it works:

  • The self-invoking function runs only once.
    It sets the counter to zero (0), and returns a function expression.
  • This way add becomes a function. The "wonderful" part is that it
    can access the counter in the parent scope.
  • This is called a JavaScript closure. It makes it possible for a function
    to have "private" variables.
  • The counter is protected by the scope of the anonymous function,
    and can only be changed using the add function.
